### Follow the below steps to run your first Rust program: 
1. Check if rustc is installed:

    ```bash
    rustc --version
    ```
2. Compile helloworld.rs

    ```bash
    rustc helloworld.rs
    ```
3. A `helloworld` binary should have been created
4. Run `helloworld`
    ```bash
    ./helloworld
    ```
